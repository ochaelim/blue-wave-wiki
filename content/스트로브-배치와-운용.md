# 스트로브 배치와 운용

> **핵심 takeaway**: 수중 조명의 핵심 원칙은 **"만질 수 없다면 빛을 줄 수 없다"**. 1m 이내 작업 거리를 유지하고, 스트로브는 **돔 중심선 위 + 가장자리 뒤쪽**에 배치하여 렌즈 앞의 물을 비추지 않는 것이 백스캐터(부유물 반사) 방지의 핵심이다.

![Strobe Positioning](https://www.uwphotographyguide.com/wp-content/uploads/2019/05/strobe-position.jpg)

## 암(Arm) 길이 기본 세팅

- 하우징 연결 암: **5~6인치**
- 스트로브 연결 암: **8~9인치**
- 이 조합이 매크로/광각/CFWA 모두를 커버하는 가장 유연한 구성

## 스트로브 배치 기본 원칙

### 위치

1. **돔 중심선 위**에 스트로브 배치 — 위에서 아래로 비추는 것이 자연스러움
2. **돔 가장자리 뒤쪽**에 위치 — 렌즈 앞 물을 직접 비추지 않기 위함
3. 앞을 향하거나 **살짝 바깥쪽** 방향

### 거리 조절

| 촬영 상황 | 스트로브 거리 |
|---|---|
| 가까운 피사체 | 스트로브를 **가깝게** 배치 |
| 먼 피사체 | 스트로브를 **멀리** 배치 |
| 부유물 많은 환경 | 스트로브를 **바깥쪽**으로 향하게 |

### 핵심 규칙

- 스트로브와 피사체를 **카메라에서 같은 거리**로 유지
- **렌즈와 피사체 사이의 물을 비추지 말 것** → 백스캐터(Backscatter) 발생
- 작업 거리는 **1m 이내** 유지 ("만질 수 없다면 빛을 줄 수 없다")

## 렌즈별 포지셔닝 상세

### 피쉬아이(10mm) 광각

![Wide angle strobe position - fisheye 10mm](https://www.uwphotographyguide.com/wp-content/uploads/2009/03/483961777_Xyw7r-M.jpg)

- 스트로브를 **뒤로 당기고 바깥쪽을 향하게** 배치
- 피사체가 너무 가까우면 중앙이 어두워질 수 있음 → 스트로브를 안쪽으로 조금 당겨서 보완

### 피쉬아이(17mm) 클로즈업

![Wide angle strobe position - close focus](https://www.uwphotographyguide.com/wp-content/uploads/2009/03/508474573_o73Yj-M.jpg)

- 피사체가 돔 포트에 매우 가까울 때: 스트로브를 **돔 포트 뒤쪽으로 더 당기고 하우징에 바짝** 붙임
- 피사체가 돔에 가까울수록 스트로브도 하우징 가까이

### 매크로 어류 촬영

![Macro strobe position - fish](https://www.uwphotographyguide.com/wp-content/uploads/2009/03/483962669_HVbwu-M.jpg)

- 스트로브를 **양옆으로 벌려** 백스캐터 최소화
- 정면에서 고르게 조명 → 어류의 색감과 패턴 균일하게 재현

### 포지셔닝 요약표

| 상황 | 스트로브 위치 |
|------|--------------|
| 광각 일반 | 양옆 45°, 뒤로 당겨 바깥 향함 |
| 광각 클로즈업 | 뒤로 최대한 당기고 하우징 가까이 |
| 매크로 균일 조명 | 양옆 평행, 약간 아래 향함 |
| 매크로 드라마틱 | 한쪽을 비스듬히 위에서 → 그림자 생성 |
| 백스캐터 심할 때 | 스트로브 더 뒤로, 더 바깥으로 |

## TTL vs Manual(수동) 모드

### TTL (자동 스트로브 레벨 조정)

- 자동으로 발광량 조절
- **다양한 상황에 유리**, 특히 가까운 거리
- 가끔 **제대로 작동 안 할 때 있음**
- 카메라 플래시를 **Fill-In(강제 발광)** 으로 설정

### Manual(수동)

- **완벽한 조정 가능**, 사용자 재량에 따라 결과 차이 큼
- 카메라 플래시를 **Manual Value (1/64)** 로 설정

#### 수동이 더 나은 경우

- **역광** 등 다이나믹 레인지가 높을 때
- 다른 피사체가 **서로 다른 거리**에 있을 때 (TTL은 양쪽 같은 출력)
- 스트로브 범위보다 **멀리** 있을 때 (최대 발광 필요)

#### 수동 광량 조절 방법

- **가까우면 줄이고, 멀면 크게** 설정
- **1/2 파워로 시작** → 한두 장 테스트 → 조정
- Shoot → Review → Adjust → Repeat

## 와이드 vs 마크로 스트로브 운용

| 항목 | 와이드 촬영 | 마크로 촬영 |
|---|---|---|
| 스트로브 수 | **이중 스트로브** | 싱글 또는 이중 |
| 시작 파워 | **1/2** | **TTL** 또는 1/2 |
| 빔 타입 | 넓은 빔 (Wide Beam) | 좁은 빔 (Narrow Beam) |
| 핵심 목적 | 전경/배경 분리 | 피사체 강조 + 어두운 배경 |
| 줌 | **사용 금지** (핀으로 이동) | 필요에 따라 1.2x~4x |

## 노출 분리 전략 (2단계 노출)

와이드 촬영에서 가장 중요한 개념:

1. **배경 노출** = [[노출보정]]으로 조정 → -1.0 ~ -2.0으로 **파란 배경** 만들기
2. **전경 노출** = 스트로브 파워로 조정 → 피사체만 밝히기

이 두 가지를 독립적으로 조절하면 **전경은 밝고 선명하게, 배경은 깊은 파란색**으로 분리된다.

## 초점 맞추기 워크플로우

1. 피사체의 **눈 부분**을 화면 가운데에 놓기
2. **반셔터** 눌러 초점 유지
3. **구도 맞추어** 셔터 완전히 눌러 촬영
4. **작업 거리 동일하게 유지** (초점 잡고 나서 앞뒤로 움직이지 않기)

## 연사 주의

- 스트로브 사용 시 **Single(단일 촬영)** 로 설정
- 연사 시 스트로브가 **충전 속도를 못 따라감**
