# 노출 보정 (Exposure Compensation)

> **핵심 takeaway**: 수중에서 노출 보정은 필수 스킬이다. 물속은 빛이 부족해 카메라가 과다 노출하려는 경향이 있고, 밝은 모래 바닥에서는 반대로 어둡게 찍힌다. **+/- 버튼 하나로 즉시 조절** 가능하니, 수중에서 글러브 끼고도 빠르게 대응할 수 있다.

## 기본 개념

- **양수(+)**: 사진이 밝아짐
- **음수(-)**: 사진이 어두워짐
- 조절 범위: **±2.0 EV**

## 조작 방법

### TG-6 (한글)

| 촬영 모드 | 방법 |
|---|---|
| P / C1/C2 / n / ^ / k | 컨트롤 다이얼 회전, 또는 F(노출보정) 버튼 → HI |
| A 모드 | F(노출보정) 버튼 → 컨트롤 다이얼 또는 HI |

### TG-7 (영어)

| Shooting Mode(촬영 모드) | Method(방법) |
|---|---|
| P / C1/C2 / n / 7 / 6 | Control Dial(컨트롤 다이얼) 회전, 또는 F(F) 버튼 → HI |
| A Mode(A 모드) | F(F) 버튼 → Control Dial(컨트롤 다이얼) 또는 HI |

**참고**: AUTO(자동) 모드와 SCN(장면) 모드에서는 노출 보정 불가.

## 수중 활용 가이드

| 상황 | 보정값 | 이유 |
|---|---|---|
| 밝은 모래 바닥 | **+0.5 ~ +1.0** | 카메라가 밝은 배경에 속아 어둡게 찍음 |
| 어두운 암반/동굴 | **-0.5 ~ -1.0** | 카메라가 밝게 올려서 배경이 뿌옇게 됨 |
| 역광 (태양 방향) | **+1.0 ~ +1.5** | 피사체가 실루엣으로 빠지는 것 방지 |
| 플래시 사용 마크로 | **-0.3 ~ -0.7** | 배경을 어둡게 만들어 피사체 강조 |
| 블루워터 배경 | **-0.3 ~ -0.5** | 파란 물색을 더 진하고 선명하게 |
