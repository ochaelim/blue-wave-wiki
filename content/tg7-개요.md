# TG-7 개요

> **핵심 takeaway**: TG-7은 TG-6의 후속 모델로 USB-C 충전, Bluetooth 지원 등이 추가되었다. **한글 메뉴를 지원하지 않으므로** 영어 메뉴명을 익혀야 한다. 수중 촬영 핵심 기능은 TG-6와 동일하지만, Pro Capture(프로 캡처) 기능과 개선된 연결성이 장점이다.

## 기본 정보

- **모델번호**: IM032
- **센서**: 1/2.33인치 CMOS (TG-6와 동일)
- **렌즈**: 25-100mm (35mm 환산), F2.0-4.9 (TG-6와 동일)
- **방수**: 15m (IPX8)
- **내충격**: 2.1m 낙하
- **내한**: -10°C
- **배터리**: LI-92B (TG-6와 동일)
- **저장매체**: SD/SDHC/SDXC
- **충전시간**: 약 3시간
- **USB**: Type-C (CB-USB14) ← TG-6과 다름
- **연결**: Wi-Fi + Bluetooth (TG-6는 Wi-Fi만)
- **언어**: 한글 미지원

## Shooting Modes(촬영 모드)

| Mode Dial(모드 다이얼) | 이름 | 설명 |
|---|---|---|
| AUTO (0) | Auto Mode(자동 모드) | 장면 자동 인식 |
| SCN | Scene Mode(장면 모드) | 프리셋 장면 선택 |
| 7 (Microscope) | Microscope Mode(현미경 모드) | 최대 1cm 근접 촬영 |
| 6 (Underwater) | Underwater Mode(수중 모드) | 수중 전용 하위 모드 5종 |
| P | Program AE(프로그램 AE) | 조리개+셔터 자동 |
| A | Aperture Priority(조리개 우선) | 조리개 수동, 셔터 자동 |
| C1/C2 | Custom Mode(사용자 정의) | 설정 저장/불러오기 |
| n | Movie Mode(동영상 모드) | 동영상 전용 |

## 주요 버튼 배치

- **Mode Dial(모드 다이얼)**: 촬영 모드 전환
- **Shutter Button(셔터 버튼)**: 반누름(AF) → 완전 누름(촬영)
- **R Button(동영상 버튼)**: 동영상 시작/종료
- **INFO Button(정보 버튼)**: 표시 전환 (Basic → Image Only → Histogram → Level Gauge)
- **F(위쪽)**: Exposure Compensation(노출 보정)
- **I(오른쪽)**: Flash(플래시) 설정
- **G(아래쪽)**: Sequential/Self-timer(연속 촬영/셀프 타이머) + Delete(삭제)
- **H(왼쪽)**: Mode Function(모드 기능) — 하위 모드 선택
- **OK Button(확인 버튼)**: Live Control(라이브 컨트롤) 표시
- **MENU/Wi-Fi**: Menu(메뉴) 표시 / Wi-Fi
- **Zoom Lever(줌 레버)**: W(Wide, 광각) ↔ T(Telephoto, 망원)
- **Control Dial(컨트롤 다이얼)**: 설정값 조절 (TG-6와 동일)
- **LOG Switch(LOG 스위치)**: GPS/센서 로그 기록 전환

## TG-6와의 주요 차이점

| 항목 | TG-6 | TG-7 |
|---|---|---|
| USB | Micro-USB | **USB Type-C** |
| Bluetooth | 미지원 | **지원** |
| 한글 메뉴 | 지원 | **미지원** |
| Pro Capture | 지원 | 지원 (개선) |
| 센서/렌즈 | 동일 | 동일 |
| 배터리 | LI-92B | LI-92B (동일) |
| HDMI | Micro | **Type-D** |
