# 노출 설정 심화 — RAW · 히스토그램 · 컬러스페이스

> RAW 촬영은 선택이 아니라 필수다. 수중에서 잃어버린 색은 후보정에서만 살릴 수 있고, RAW만이 그 여지를 준다. 히스토그램은 수중 LCD를 믿지 못할 때 유일한 노출 판단 도구.

## RAW vs JPEG

RAW는 카메라 센서가 캡처한 **모든 데이터를 압축 없이 저장**하는 포맷. TG-6/7의 RAW 확장자는 `.ORF` (Olympus Raw Format).

### RAW의 실질적 이점

**1. 비트 심도**
- JPEG: **8비트** → 채널당 256단계
- RAW: **12~14비트** → 채널당 4,096~16,384단계
- 수중처럼 색 정보가 적은 환경에서 이 차이가 후보정 품질을 결정

**2. 다이나믹 레인지**
- RAW는 **하이라이트·섀도우 각각 약 1스톱씩 추가 복원** 가능
- 어두운 산호 + 밝은 수면 동시 촬영 → RAW면 양쪽 다 살릴 수 있음

**3. 화이트밸런스 후보정**
- JPEG는 촬영 시점에 WB를 굽혀서 저장 → 나중에 바꾸면 화질 저하
- RAW는 WB가 메타데이터로만 기록 → **원하는 값으로 자유롭게** 조정
- 수중에서 정확한 WB 설정이 어려운 상황에서 핵심 이점

**4. 색공간 선택 자유**
- RAW 현상 시 sRGB / Adobe RGB 중 선택 가능

### RAW 단점 & 해결책

| 단점 | 해결책 |
|------|--------|
| 파일 크기 큼 (TG-7 기준 약 12~18MB/장) | 대용량 SD카드 (64GB 이상) |
| 바로 공유 불가 | RAW+JPEG 동시 기록 |
| 편집 소프트웨어 필요 | Lightroom, Capture One, Olympus Workspace (무료) |
| 후보정 시간 소요 | 프리셋 활용 |

### JPEG도 괜찮은 경우

- 매크로 촬영 시 스트로브 빛이 정확하면 카메라 내부 처리로도 충분
- 빠른 SNS 업로드 목적
- 카드 용량 한계 상황

### 권장 설정: RAW+JPEG 동시 기록

TG-6/7 메뉴 경로: `카메라 메뉴 → 화질 → RAW+F (또는 RAW+N)`

- RAW: [[수중-색과-후보정|후보정]]·보관용
- JPEG: 빠른 공유·확인용
- 수중에서 JPEG로 구도 확인 → 귀가 후 RAW로 정밀 보정

## 히스토그램

수중에서는 LCD 밝기가 주변 환경 때문에 실제와 다르게 보인다. 히스토그램은 노출의 객관적 지표.

### 히스토그램 읽는 법

- 가로축: 어두운 쪽(왼쪽) → 밝은 쪽(오른쪽)
- 세로축: 해당 밝기 픽셀의 수
- **왼쪽 끝에 몰림** = 언더노출 (디테일 손실)
- **오른쪽 끝에 몰림** = 오버노출/하이라이트 날아감

### 수중 사진 적용 팁

- 약간 오른쪽에 치우친 노출(ETTR)이 RAW 후보정에서 유리 — 단, 오른쪽 끝에 붙으면 안 됨
- 스트로브 사용 시 히스토그램이 급격히 변할 수 있어 촬영 후 반드시 확인
- TG-6/7: 촬영 후 재생 화면에서 `INFO` 버튼으로 히스토그램 표시 가능
- [[노출보정]]과 함께 활용

## 컬러스페이스

컬러스페이스는 사진이 표현할 수 있는 색의 범위(색역)를 정의한다.

| 컬러스페이스 | 색역 | 용도 |
|-------------|------|------|
| **sRGB** | 좁음 | 웹·SNS·일반 모니터 표준 |
| **Adobe RGB** | 넓음 | 전문 인쇄·후보정용 |
| **ProPhoto RGB** | 매우 넓음 | 고급 RAW 워크플로용 (Lightroom 내부 처리) |

### 수중 사진에서 실질적 의미

- JPEG 촬영 시: 카메라에서 선택한 컬러스페이스로 고정 → 나중에 바꾸면 색 손실
- RAW 촬영 시: 후보정 소프트웨어에서 현상 시점에 선택 가능 → 자유도 높음
- SNS·웹 업로드 최종본: **sRGB로 내보내기** 필수 (Adobe RGB를 그냥 올리면 채도가 죽어 보임)
- TG-6/7 JPEG: sRGB 고정. RAW(ORF)로 찍으면 Lightroom 등에서 선택 가능

## 노출 삼각형 수중 적용 요약

### 조리개

- f/2.8~f/4: 피사계심도 얕음, [[수중-보케|보케]]
- f/11~f/16: 전체 선명, 매크로에 유리
- 수중 광각 sweet spot: f/8~f/11
- 상세: [[조리개-작동방식]]

### 셔터스피드

- **스트로브 동조속도** 초과 금지: 1/160~1/250 이하
- 빠를수록 배경 바다 어두워짐 → 피사체 분리에 활용
- 느리면 운동 블러 (물고기, 부유물)

### ISO

- 가능한 낮게 (노이즈 최소화)
- TG-6/7: ISO 100~200에서 최고 화질
- 주변광 촬영 시 400~800까지 허용
