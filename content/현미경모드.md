# 현미경 모드 (Microscope Mode)

> **핵심 takeaway**: 현미경 모드(k / 7)는 피사체에서 **최대 1cm까지** 접근하여 촬영할 수 있는 강력한 근접 모드다. 수중에서 누디브랜치나 극소 생물을 촬영할 때는 [[수중모드]]의 Underwater Microscope(수중 현미경)를 쓰지만, 육상에서 장비 디테일 촬영이나 해양 생물 표본 촬영에는 이 모드가 유용하다.

## 모드 진입

- **TG-6**: 모드 다이얼을 **k**로 돌림
- **TG-7**: 모드 다이얼을 **7**(Microscope)로 돌림

하위 모드 전환: **H(왼쪽) 버튼** → HI로 선택 → Q/OK 버튼

## 하위 모드 4종

### 1. Microscope(현미경) — m / c

피사체에서 최대 1cm 거리에서 촬영 가능. 기본 근접 촬영 모드.

### 2. Focus Stacking(심도 합성) — o / d

카메라가 여러 이미지를 촬영하며 자동으로 초점을 이동. 피사계 심도가 큰 합성 이미지를 생성한다.

- 두 개의 이미지 기록: 첫 번째 프레임 + 합성 이미지
- 촬영 수와 대기 시간은 **Focus Stacking Settings(심도합성설정)**에서 설정
- 카메라 흔들림 시 합성 불가능할 수 있음 → 삼각대 권장

### 3. Focus Bracketing(포커스 브라케팅) — p / e

여러 이미지를 자동 초점 이동하며 촬영. Focus Stacking과 달리 카메라 내에서 합성하지 않고 개별 이미지를 저장한다. 후보정에서 직접 합성하고 싶을 때 사용.

- 이미지 수와 촬영 단계는 **Focus BKT** 메뉴에서 설정

### 4. Microscope Control(현미경 컨트롤) — n / f

모니터에서 피사체를 확대하여 볼 수 있는 모드. 촬영 거리 1cm일 때의 배율 수준이 표시된다.

- **I 버튼**: 2배 줌 → 4배 줌 → 줌 해제 순환
- 배율이 높을수록 이미지가 거칠어질 수 있음
- 촬영 중 줌 배율이 항상 표시됨

## 촬영 팁

- 같은 초점 거리에서 연속 촬영: **AFL(초점 잠금)** 또는 **MF(수동 초점)** 사용
- 심도 합성 촬영 시 반드시 **삼각대** 사용
- [[노출보정]]으로 밝기 미세 조정 가능
