# 촬영 모드 비교 (TG-6 / TG-7)

> **핵심 takeaway**: TG-6와 TG-7의 촬영 모드는 기능적으로 거의 동일하다. 다이빙 강사로서 가장 많이 쓰게 될 모드는 **수중 모드(^/6)**, **현미경 모드(k/7)**, 그리고 **P 모드**다. 수중에서는 수중 모드를 기본으로, 특별한 세팅이 필요하면 P 모드 + [[노출보정]] 조합을 쓴다.

## 모드별 비교표

| 기능 | TG-6 (한글) | TG-7 (영어) | 용도 |
|---|---|---|---|
| 자동 | B (AUTO) | AUTO (0) | 초보자/급할 때 |
| 장면 | SCN | SCN | 상황별 프리셋 |
| 현미경 | k | 7 (Microscope) | 1cm 초근접 |
| 수중 | ^ | 6 (Underwater) | 수중 전용 |
| 프로그램 | P | P (Program AE) | 범용, 노출보정 가능 |
| 조리개 우선 | A | A (Aperture Priority) | 심도 조절 |
| 사용자 정의 | C1/C2 | C1/C2 (Custom) | 자주 쓰는 설정 저장 |
| 동영상 | n | n (Movie) | 동영상 전용 |

## 다이빙 강사 추천 세팅

### 수중 촬영 기본
1. 모드 다이얼 → **^(수중) / 6(Underwater)**
2. 상황에 맞는 하위 모드 선택 → [[수중모드]] 참조
3. [[노출보정]]으로 밝기 미세 조정

### 수중 마크로/누디 촬영
1. 수중 모드 → **Underwater Microscope(수중 현미경)**
2. 또는 [[현미경모드]] → Microscope Control(현미경 컨트롤)로 2x/4x 확대

### 자주 쓰는 설정 저장
1. 원하는 설정 완료 후 → 메뉴 → 촬영 메뉴 1 → **재설정/사용자 정의 등록(Reset/Assign Custom Modes)**
2. C1 또는 C2에 저장
3. 모드 다이얼을 C1/C2로 돌려 즉시 불러오기
