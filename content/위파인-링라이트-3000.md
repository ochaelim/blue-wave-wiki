# 위파인 링라이트 3000 (Weefine Ring Light 3000)

> **핵심 takeaway**: 링라이트 3000은 TG-6/TG-7에 직접 장착하는 수중 링라이트로, 연속 조명(1800루멘)과 스트로브(3000루멘)를 모두 지원한다. 마크로 촬영에서 **균일한 조명**을 제공하며, 100m 수심까지 사용 가능하다.

## 기본 스펙

| 항목 | 사양 |
|---|---|
| 조명 모드 | 연속 조명 + 스트로브 조명 |
| 광색 | 백색광 / 적색광 / 청색광 |
| 연속 조명 밝기 | **최대 1800루멘** |
| 스트로브 밝기 | **최대 3000루멘** |
| 수심 등급 | **100m / 330ft** |
| 빔 각도 | 육지 100° / 수중 90° |
| 색온도 | 5000~5500K |
| 파워 레벨 | 100% / 75% / 50% / 25% |
| 버튼 | 듀얼 푸시 버튼 (배터리 레벨 미터 포함) |
| 사용 시간 | 연속 점등 약 **45분** / 스트로브 약 **12시간** |
| 배터리 | 26650 리튬 배터리 1개 |
| 충전 시간 | 약 4시간 (5V/2A) |
| 크기 | 123.8 × 159.6 × 40.6mm |
| 마운트 | **M67** |

## 조명 모드 선택

### 연속 조명 모드

- 영상 촬영에 적합
- 초점 맞추기 쉬움 (실시간 조명)
- 최대 45분 사용 → 다이빙 1회분
- [[마크로-영상-세팅]] 참조

### 스트로브 모드

- 사진 촬영에 적합
- 3000루멘으로 순간 강한 빛 제공
- 12시간 사용 가능 (충전 사이 여유)
- [[마크로-사진-세팅]] 참조

### 적색광 / 청색광

- **적색광**: 야행성 생물 관찰용 (생물에게 방해 최소화)
- **청색광**: 형광 촬영용

## TG-6/TG-7 연동 세팅

스트로브 모드 사용 시:
- 카메라 플래시 → **Fill-In(강제 발광)** (TTL 모드 시)
- 카메라 플래시 → **Manual Value 1/64** (수동 모드 시)
- 연사 → **Single(단일 촬영)** (스트로브가 연사를 못 따라감)

연속 조명 모드 사용 시:
- 카메라 플래시 → **Off(발광 금지)**
- [[화이트밸런스]] → **Auto** (백색광이므로 자동이 적합)
