# 마크로 영상 세팅 (TG-6 / TG-7 공통)

> **핵심 takeaway**: 수중 마크로 영상의 핵심은 **삼각대(또는 안정적 자세) + 좁은 빔의 싱글 라이트 + 눈에 초점**이다. 사진과 달리 영상은 **흔들림이 치명적**이므로 삼각대 사용을 권장하고, 작업 거리 30cm 이내에서 촬영한다.

## 필수 장비

- **트레이**: 삼각대(Tripod) 권장
- **조명**: 싱글 비디오 라이트 (**Narrow Beam**, 좁은 빔)

## 보트 위 세팅 (Set & Forget)

| 설정 항목 | TG-7 (영어) | TG-6 (한글) | 값 |
|---|---|---|---|
| 리셋 | Full Reset | 전체 리셋 | **먼저 실행** |
| 모드 | P Mode | P 모드 | **P 모드** |
| 노출 보정 | Exposure Comp | 노출보정 | **-0.7** |
| Live View Boost | Live View Boost | 라이브 뷰 부스트 | **ON** |
| 초점 모드 | Focus Mode | AF 모드 | **AF Super Macro** |
| 비트레이트 | Video Bit Rate | 비디오 비트레이트 | **Super Fine** |
| 해상도/프레임 | Resolution & Framerate | 해상도/프레임레이트 | **1080HD 60p** |
| 화면비 | Aspect Ratio | 종횡비 | **16:9** |
| 플래시 | Flash | 플래시 | **Off** |

## 입수 후 조정

| 항목 | 값 | 비고 |
|---|---|---|
| 화이트밸런스 | **Auto** | 라이트 사용 시 자동이 적합 |
| 라이트 파워 | **Full** → 하이라이트 보고 조정 | 최대 출력 시작 |
| 라이트 위치 | **피사체 위쪽, 아래+앞 방향** | 위에서 비추는 것이 자연스러움 |
| 작업 거리 | **0~30cm** | 가까울수록 좋다 |
| 줌 | **4x 이하** (피사체에 따라) | |
| AF | **눈에 초점** | 반셔터로 초점 잡기 |
| 삼각대 | **사용 권장** | 안정성이 핵심 |

## 마크로 영상 핵심 노트

- **항상 눈에 선명한 초점**을 맞춰라
- 라이트로 피사체를 밝혀 **돋보이게** 만들어라
- **어두운 배경 유지**를 시도하라
- 렌즈와 피사체 사이의 물을 비추지 말 것 → [[스트로브-배치와-운용]] 참조 (백스캐터 방지)

## 초간단 세팅 (초보용)

**Fish Mode: Macro** 또는 **Microscope** (간편 프리셋)
