# Pro Capture(프로 캡처)

> **핵심 takeaway**: 프로 캡처는 셔터 버튼을 완전히 누르기 **0.5초 전부터** 이미 촬영을 시작하는 기능이다. 수중에서 순간적으로 나타나는 생물이나 먹이 활동 장면처럼 **타이밍이 생명인 순간**을 포착하는 데 매우 유용하다.

## 작동 원리

```
반누름 시작 ──────────── 완전 누름 ──────── 촬영 종료
     │                        │
     ▼                        ▼
 [버퍼에 연속 촬영 중]     [카드에 기록 시작]
 (최대 5프레임 소급)      (연속 촬영 계속)
```

1. 셔터 버튼 **반누름** → 카메라가 약 10fps로 버퍼에 촬영 시작
2. 셔터 버튼 **완전 누름** → 그 순간부터 카드에 기록 + 반누름 중 찍힌 **최대 5프레임도 소급 저장**
3. 셔터에서 손을 떼면 촬영 종료

## 설정 방법

1. **G(아래쪽) 버튼** (gY 버튼) 누름
2. **HI**로 **[s] Pro Capture(프로 캡처)** 선택
3. **Q/OK** 확인
4. 촬영 시 셔터 반누름 → 완전 누름

## 주의사항

- 반누름 상태에서 **최대 1분간** 버퍼 촬영 지속 → 이후 다시 반누름 필요
- **플래시 사용 불가** (Flash disabled)
- 형광등 아래에서 **플리커 왜곡** 발생 가능
- 반누름 중에는 **전자 셔터음 없음**, 화면도 어두워지지 않음
- 초점, 노출, 화이트밸런스는 **첫 프레임 기준**으로 고정
- 가장 느린 셔터 속도에 제한 있음

## 수중 활용

| 상황 | 효과 |
|---|---|
| 물고기 먹이 활동 | 정확한 입 벌리는 순간 포착 |
| 갑각류 탈피/이동 | 빠른 움직임 순간 캡처 |
| 거북이/가오리 접근 | 접근 순간의 최적 구도 확보 |
| 청소새우 활동 | 미세한 행동 순간 기록 |
