# 조리개 작동 방식 (TG-6 / TG-7 공통)

> **핵심 takeaway**: TG-6/TG-7의 물리적 조리개는 **f/2.0과 f/2.8 딱 2개**뿐이다. f/2.8 이상의 값은 **ND 필터로 흉내**내는 것이므로 심도는 전혀 변하지 않는다. 하지만 수중 스트로브 촬영에서 **검은 배경 효과**를 만들 때는 이 "가짜 조리개"가 유용하다.

## 실제 물리적 조리개: 2개뿐

| 줌 위치 | 물리적 조리개 |
|---|---|
| 25mm (광각 끝) | **f/2.0**, **f/2.8** |
| 100mm (망원 끝) | 렌즈 구조에 따라 제한됨 |

이 카메라는 1/2.33인치의 작은 센서를 사용하기 때문에 회절(Diffraction) 현상이 쉽게 발생한다. 조리개를 실제로 조이면 회절로 인해 오히려 화질이 떨어지므로, 물리적 조리개를 2개로 제한한 것이다.

## ND 필터로 조리개를 "흉내" 낸다

A Mode(조리개 우선 모드)에서 f/2.8 이상의 값(f/4, f/5.6, f/8 등)을 선택하면:

```
실제 일어나는 일:
조리개 = f/2.8 고정 (물리적으로 변하지 않음)
+ 내장 ND 필터(중성 밀도 필터)가 빛 경로에 삽입
= 센서에 도달하는 빛의 양만 줄어듦
```

### 결과

| 항목 | f/2.8 (실제) | f/8 (ND 시뮬레이션) |
|---|---|---|
| 빛의 양 | 많음 | 적음 (ND 필터로 감소) |
| 심도(피사계 심도) | 얕음 | **얕음 (변화 없음!)** |
| 셔터 속도 | 빠름 | **느려짐** |
| ISO | 낮게 유지 가능 | **올라갈 수 있음** |
| 회절 | 없음 | 없음 (물리적으로 f/2.8이므로) |

### 구체적 예시: f/8 설정

```
f/8 설정 = f/2.8 (실제 조리개) + 3스톱 ND 필터
         f/2.8 → f/4 → f/5.6 → f/8 = 3스톱 차이
```

## 핵심 함정 — 단점만 있고 장점은 없다

| | 일반 카메라에서 f/8 | TG-6/TG-7에서 f/8 |
|---|---|---|
| 심도 깊어짐 | **O** | **X** (f/2.8 그대로) |
| 셔터 느려짐 | O | **O** (동일하게 발생) |
| ISO 올라감 | O | **O** (동일하게 발생) |

**단점(느린 셔터, 높은 ISO)은 그대로 생기지만, 장점(깊은 심도)은 전혀 없다.**

### 실험 결과

- **f/6.3**: 심도가 살짝 늘어남 (미미한 차이)
- **f/18**: 심도가 더 늘어나지 않고 **오히려 나빠 보임**

→ 심도를 늘리고 싶으면 높은 f값 대신 [[현미경모드]]의 **Focus Stacking(심도 합성)** 사용

## 그럼 f/8은 언제 쓸모 있나? — 수중 검은 배경 효과

스트로브 촬영에서 **배경을 새까맣게 날리고 싶을 때** 유용하다:

```
f/8 (ND로 주변광 차단) + 스트로브 (피사체만 밝힘)
= 배경은 검게, 피사체만 밝게 = 극적인 마크로 사진
```

1. f/8로 설정 → 주변광(자연광)이 크게 줄어들어 **배경이 검어짐**
2. 스트로브가 충분한 광량으로 피사체를 비춤 → **피사체는 정상 노출**
3. 결과: 피사체만 떠오르는 **검은 배경 마크로**

이것이 [[마크로-사진-세팅]]에서 **f/9~f/18 + 노출 보정 -2.0**을 권장하는 이유다. 심도 효과가 아니라 **배경 제거 효과**를 노리는 것이다.

## 실전 권장 세팅

| 목적 | 추천 f값 | 이유 |
|---|---|---|
| 밝은 환경 + 얕은 심도 원할 때 | **f/2.0** | 최대 개방, 가장 빠른 셔터 |
| 범용 촬영 | **f/2.8** | 실제 조리개의 마지막 값, 심도 약간 증가 |
| 스트로브 마크로 (검은 배경) | **f/8~f/14** | ND로 주변광 차단, 스트로브로 피사체만 노출 |
| 와이드 + 스트로브 | **f/2.8** | 빛이 부족한 수중에서 최대한 밝게 |
