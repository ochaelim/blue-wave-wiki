# 와이드 촬영 세팅 (TG-6 / TG-7 공통)

> **핵심 takeaway**: 와이드 촬영의 3대 원칙: **가까이, 더 가까이 가라** / **전경(가까운 피사체)에 초점** / **스트로브로 전경과 배경을 분리**하라. 촬영 중 조정할 것은 딱 2가지뿐: 노출 보정(배경 파란색 농도)과 스트로브 파워(전경 밝기).

## 필수 장비

| 항목 | 사진 촬영 | 영상 촬영 |
|---|---|---|
| 렌즈 | **M52 Wide Angle Lens** (광각 컨버전 렌즈) | M52 Wide Angle 또는 Air Lens |
| 조명 | **이중 스트로브** (Dual Strobes) | **이중 비디오 라이트** (Wide Beam) |
| 트레이 | — | **Double Handle Tray** (양손 트레이) |

## 와이드 사진 (스트로브) — 보트 위 세팅

| 설정 항목 | TG-7 (영어) | TG-6 (한글) | 값 |
|---|---|---|---|
| 모드 | A (Aperture Priority) | A (조리개 우선) | **A 모드** |
| 조리개 | Aperture | 조리개 | **f/2.8** (중간값) |
| ISO | ISO Auto | ISO 자동 | **Auto (100~400 제한)** |
| 최소 셔터 속도 | Min Shutter Speed | 최소 셔터 속도 | **1/125** |
| Live View Boost | Live View Boost | 라이브 뷰 부스트 | **ON** |
| 화이트밸런스 | White Balance | 화이트밸런스 | **Auto** |
| 초점 모드 | Focus Mode | AF 모드 | **AF** |
| 플래시 | Flash | 플래시 | **Fill-In** 또는 **Manual 1/64** |
| 손떨림 보정 | Stabilization | 손떨림 보정 | **On** |
| 연사 | Frame Rate | 연사 | **Single** (단일) |
| 파일 형식 | Quality | 화질 | **RAW** 또는 **RAW+JPEG "LF"** |

### 입수 후 조정

| 항목 | 값 | 비고 |
|---|---|---|
| 노출 보정 | **-1.0 ~ -2.0** | 파란 배경 농도 조절용 |
| 작업 거리 | **10cm ~ 1m** | 만질 수 없으면 빛을 줄 수 없다 |
| 줌 | **사용 금지** | 발로 이동해서 구도 조정 |
| 초점 영역 | **가장 가까운 전경 요소** | 전경에 초점 필수 |
| 스트로브 파워 | **1/2에서 시작** → 조정 | 테스트 후 미세 조정 |

## 와이드 영상 (비디오 라이트) — 보트 위 세팅

| 설정 항목 | TG-7 (영어) | TG-6 (한글) | 값 |
|---|---|---|---|
| 리셋 | Full Reset | 전체 리셋 | **먼저 실행** |
| 모드 | P Mode | P 모드 | **P 모드** |
| 노출 보정 | Exposure Comp | 노출보정 | **-0.7** |
| Live View Boost | Live View Boost | 라이브 뷰 부스트 | **ON** |
| 초점 모드 | Focus Mode | AF 모드 | **AF** |
| 비트레이트 | Video Bit Rate | 비디오 비트레이트 | **Super Fine** |
| 해상도/프레임 | Resolution & Framerate | 해상도/프레임레이트 | **1080HD 60p** |
| 화면비 | Aspect Ratio | 종횡비 | **16:9** |
| 플래시 | Flash | 플래시 | **Off** |

### 입수 후 조정 (영상)

| 항목 | 값 | 비고 |
|---|---|---|
| 화이트밸런스 | **Custom WB** (수동) | **수심 3m마다 재측정** |
| 작업 거리 | **10cm ~ 1m** | 만질 수 없으면 빛을 줄 수 없다 |
| AF | 프레임 내 가장 가까운 요소 | 전경 초점 |
| 라이트 파워 | **Full** → 하이라이트 보고 조정 | 최대 출력에서 시작 |
| 라이트 위치 | 돔 중심선 위, 가장자리 뒤쪽 | 앞으로 또는 살짝 바깥 방향 |
| 돔 관리 | **입수 시 돔에 버블 제거 (Burp)** | 기포가 화질에 영향 |
| 안정성 | **촬영 중 안정 유지** | 영상은 흔들림이 치명적 |

## 와이드 촬영 핵심 노트

- **가까이 가라, 그리고 더 가까이 가라** — 줌 대신 핀으로 이동
- **가장 가까운 전경 요소에 초점**
- 수중에서 조정할 건 **딱 2가지**: 노출 보정(배경) + 스트로브 파워(전경)
- 노출 보정을 내리면 **더 파란 배경**, 올리면 **시안(cyan) 톤**
- 스트로브로 **전경과 배경을 분리** — 전경만 밝히고 배경은 자연광
- 스트로브와 피사체를 **카메라에서 같은 거리**로 유지 → 렌즈 앞 물을 비추지 않음
- 영상은 **선명한 초점 + 정확한 색감 + 안정성** 3박자
