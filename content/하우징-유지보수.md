# 하우징 유지보수 & 관리

> O링 하나의 실수가 카메라를 수장시킨다. 입수 전 체크리스트를 매번 따르고, 다이빙 후 민물 세척을 건너뛰지 마라. 여행 시 온도 변화 적응 시간(2~3시간)도 반드시 확보.

![Housing Maintenance](https://www.uwphotographyguide.com/wp-content/uploads/2024/02/setup.jpg)

---

## 입수 전 체크리스트

- [ ] O링 상태 확인 — 이물질(모래, 머리카락) 없는지
- [ ] O링 그리스 상태 (너무 많아도 문제, 적어도 문제)
- [ ] 모든 잠금장치 닫혔는지 확인
- [ ] 수압 테스트 (얕은 수심에서 1분 대기 후 누수 확인)
- [ ] [[스트로브-선택-가이드|스트로브]] 케이블/광섬유 연결 확인
- [ ] 배터리 충전 상태 (카메라 + 스트로브)
- [ ] 메모리카드 용량 확인
- [ ] [[카메라-기본설정-체크리스트|카메라 설정]] 확인

---

## O링 관리

- O링은 [[웻렌즈-하우징-시스템|하우징]] 방수의 핵심
- **매 다이빙 후** 깨끗이 닦고 상태 확인
- **그리스 도포**: 얇고 균일하게, 손가락으로 전체에 얇게
- O링 교체 주기: 이상 없어도 **연 1회** 교체 권장
- 여분 O링 항상 다이빙백에 지참

---

## 입수 후 관리

- **민물 세척 필수** — 염분이 금속 부식 유발
- 세척 순서: 버튼 누르지 않고 민물에 10~15분 담금 → 부드럽게 세척
- 세척 후 **완전히 건조 후에야** 열기
- 포트 유리/아크릴 — 부드러운 천으로 닦기 (스크래치 주의)

---

## 장기 보관

- O링 제거 후 보관 (압착 방지)
- 서늘하고 건조한 곳 (직사광선 피하기)
- 실리카겔 동봉 권장
- 장기 미사용 시 모든 버튼/다이얼 중간 위치로

---

## 여행 시 주의사항

- 위탁 수하물 시 하드케이스 사용
- 기내 반입: 리튬 배터리 제한 확인 (스트로브 NiMH는 무방)
- 현지 도착 후 온도 변화로 인한 수분 응결 주의 → **2~3시간 적응 후 입수**

---

## TG-6/7 자체 방수 관리

TG-6/7은 하우징 없이도 15m 방수지만:
- USB/마이크 커버 반드시 닫기
- 커버 실리콘 상태 정기 확인
- Olympus 공식 방수 테스트 주기: **2년마다** 권장
- 하우징 사용 시 위 관리 불필요
