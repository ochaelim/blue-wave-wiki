# TG-6 개요

> **핵심 takeaway**: TG-6는 한글 메뉴를 지원하는 올림푸스의 방수 컴팩트 카메라로, 수중 15m까지 촬영 가능하다. 수중 전용 모드(^)와 현미경 모드(k)를 갖추고 있어 다이빙 강사가 현장에서 바로 활용하기에 최적화된 장비다. TG-7의 전세대 모델이지만 핵심 기능은 거의 동일하며, **한글 UI**가 최대 장점이다.

## 기본 정보

- **모델번호**: IM015
- **센서**: 1/2.33인치 CMOS
- **렌즈**: 25-100mm (35mm 환산), F2.0-4.9
- **방수**: 15m (IPX8)
- **내충격**: 2.1m 낙하
- **내한**: -10°C
- **배터리**: LI-92B (리튬이온)
- **저장매체**: SD/SDHC/SDXC
- **충전시간**: 약 3시간
- **USB**: Micro-USB (CB-USB12)
- **언어**: 한글 지원

## 촬영 모드

| 모드 다이얼 | 이름 | 설명 |
|---|---|---|
| B (AUTO) | 자동 모드 | 카메라가 장면에 맞게 자동 설정 |
| SCN | 장면 모드 | 피사체/장면에 최적화된 프리셋 |
| k | 현미경 모드 | 최대 1cm 근접 촬영, 심도 합성 지원 |
| ^ | 수중 모드 | 수중 전용 하위 모드 5종 |
| P | 프로그램 AE | 조리개+셔터 자동, 노출 보정 가능 |
| A | 조리개 우선 | 조리개 수동 선택, 셔터 자동 |
| C1/C2 | 사용자 정의 | 설정 저장/불러오기 |
| n | 동영상 | 동영상 전용 모드 |

## 주요 버튼 배치

- **모드 다이얼**: 촬영 모드 전환
- **셔터 버튼**: 반누름(AF) → 완전 누름(촬영)
- **R 버튼**: 동영상 촬영 시작/종료
- **INFO 버튼**: 정보 표시 전환 (기본 → Off → 히스토그램 → 수준기)
- **F(위쪽)**: 노출 보정
- **I(오른쪽)**: 플래시 설정
- **G(아래쪽)**: 연속 촬영/셀프 타이머
- **H(왼쪽)**: 모드 기능 (하위 모드 선택)
- **Q 버튼**: 라이브 컨트롤 / 확인
- **MENU/Wi-Fi**: 메뉴 표시 / Wi-Fi 연결
- **줌 레버**: W(광각) ↔ T(망원)
