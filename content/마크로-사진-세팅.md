# 마크로 사진 세팅 (TG-6 / TG-7 공통)

> **핵심 takeaway**: 수중 마크로 사진의 핵심은 **스트로브(또는 비디오 라이트) + 어두운 배경 + 눈에 초점**이다. 골프공보다 작은 피사체를 찾아 1~15cm 거리에서 촬영하며, 노출 보정 -2.0으로 배경을 날리고 스트로브로 피사체만 밝힌다.

## 스트로브 사용 마크로 — 보트 위 세팅 (Set & Forget)

### 카메라 설정

| 설정 항목 | TG-7 (영어) | TG-6 (한글) | 값 |
|---|---|---|---|
| 모드 | A (Aperture Priority) | A (조리개 우선) | **A 모드** |
| 조리개 | Aperture | 조리개 | **f/9 ~ f/18** (줌에 따라) |
| 노출 보정 | Exposure Comp | 노출보정 | **-2.0** |
| ISO | ISO | ISO | **100** |
| Live View Boost | Live View Boost | 라이브 뷰 부스트 | **ON** (LCD 밝게) |
| 화이트밸런스 | White Balance | 화이트밸런스 | **Auto** |
| 초점 모드 | Focus Mode | AF 모드 | **AF Super Macro** |
| 파일 형식 | Quality | 화질 | **RAW** 또는 **RAW+JPEG "LF"** |
| 플래시 | Flash | 플래시 | **Fill-In** (강제 발광) |
| 스트로브 모드 | — | — | **TTL** |

### 입수 후 조정

| 항목 | 값 | 비고 |
|---|---|---|
| 작업 거리 | **2.5~15cm** (최대 30cm) | 가까울수록 좋다 |
| 줌 | **1.2x ~ 4x** | 줌으로 배율 조절 |
| 초점 영역 | 눈, 얼굴, 핵심 디테일 | 반드시 눈에 초점 |
| 스트로브 위치 | **위쪽** (그림자+어두운 배경) | 위에서 아래로 비추는 것이 자연스러움 |
| 스트로브 거리 | 피사체~렌즈 거리와 동일하게 | 물속 부유물 반사 방지 |

## 비디오 라이트 사용 마크로 — 보트 위 세팅

| 설정 항목 | TG-7 (영어) | TG-6 (한글) | 값 |
|---|---|---|---|
| 모드 | Microscope / P | 현미경 / P | **Microscope(현미경)** 또는 **P** |
| 노출 보정 | Exposure Comp | 노출보정 | **-0.3** |
| ISO | ISO | ISO | **100~400** (최대한 낮게) |
| Live View Boost | Live View Boost | 라이브 뷰 부스트 | **ON** |
| 초점 | Focus | AF 모드 | **Autofocus** |
| 플래시 | Flash | 플래시 | **Off** (발광 금지) |
| 손떨림 보정 | Stabilization | 손떨림 보정 | **On** |
| 연사 | Frame Rate | 연사 | **Single~High** |
| 화이트밸런스 | White Balance | 화이트밸런스 | **Auto** |
| 파일 형식 | File Type | 화질 | **RAW / RAW+JPEG** |

### 입수 후 조정 (비디오 라이트)

| 항목 | 값 |
|---|---|
| 셔터 속도 | **1/250 이상** (빠르게) |
| 작업 거리 | **2.5~15cm** |
| 줌 | 필요에 따라 |
| 초점 영역 | 눈 (Critical Detail Area) |
| 라이트 파워 | **최대 밝기** |
| 라이트 위치 | **앞쪽 & 아래 방향** |

## 마크로 촬영 핵심 노트

- 골프공보다 **작은 피사체**를 찾아라
- 반드시 **눈, 얼굴, 핵심 부위에 선명한 초점**
- **어두운 배경**이 피사체를 돋보이게 한다
- 아래쪽 **그림자**가 대비와 입체감을 준다
- **Shoot → Review → Adjust → Repeat** (촬영→확인→조정→반복)

## 초간단 세팅 (초보용)

스트로브 사용 시:
1. [[수중모드]] → **Underwater Macro(수중 마크로)** 선택
2. 스트로브를 **TTL** 모드로

비디오 라이트 사용 시:
1. [[현미경모드]] → **Microscope(현미경)** 선택
2. 라이트 최대 밝기
