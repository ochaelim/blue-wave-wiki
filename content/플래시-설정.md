# 플래시 설정 (Flash Settings)

> **핵심 takeaway**: 수중에서 플래시는 **색감 복원**의 핵심이다. 물은 빨간색 빛을 흡수하므로, 플래시 없이는 모든 것이 파랗게/초록색으로 보인다. 마크로 촬영에서는 내장 플래시만으로도 충분하지만, 와이드 촬영에는 외장 수중 스트로브가 필요하다.

## Flash Mode(플래시 모드) 종류

| TG-7 영어 | 기호 | TG-6 한글 | 설명 |
|---|---|---|---|
| Flash Auto | AUTO | 플래시 자동 | 어두울 때만 자동 발광 |
| Fill In | A | 강제 발광 | 항상 발광 |
| Red-eye | e | 적목 감소 | 적목 현상 방지 선발광 |
| Flash Off | B | 발광 금지 | 발광 안 함 |
| Slow Sync + Red-eye | f | 슬로우 싱크 + 적목 | 느린 셔터 + 적목 감소 |
| Slow Sync (1st Curtain) | g | 슬로우 싱크 (선막) | 느린 셔터로 배경 밝게 |
| Manual Value | h, i 등 | 수동 발광량 | 발광량 직접 조절 |
| Remote Control | j | 리모트 컨트롤 | 수중 외장 플래시/RC 플래시 |
| LED On | k | LED 켜기 | LED 보조광으로 촬영 |

## 조작 방법

1. **I(오른쪽) 버튼** 누름 → Flash(플래시) 설정 화면
2. **HI** 또는 컨트롤 다이얼로 모드 선택
3. **Q/OK 버튼**으로 확인

### Manual Value(수동 발광량) 설정
- 모드 선택 화면에서 [Manual Value]를 선택한 상태에서 **INFO 버튼** → FGHI로 발광량 조절

## 수중 촬영 플래시 가이드

| 상황          | 추천 설정                                      | 이유               |
| ----------- | ------------------------------------------ | ---------------- |
| 수중 마크로 (근접) | **Fill In(강제 발광)** 또는 **Manual Value(수동)** | 색감 복원, 발광량 미세 조절 |
| 수중 와이드 (광각) | **Flash Off(발광 금지)** + 외장 스트로브             | 내장 플래시 범위 부족     |
| 수중 스냅샷      | **Flash Auto(자동)**                         | 상황에 따라 자동 판단     |
| 블랙워터 다이빙    | **Manual Value(수동)** 낮은 출력                 | 부유물 반사(백스캐터) 최소화 |
|             |                                            |                  |

## 주의사항

- **Red-eye(적목 감소)** 모드는 선발광 후 약 1초 딜레이가 있음 — 움직이는 생물에 부적합
- **Slow Sync(슬로우 싱크)**는 셔터 속도가 느려지므로 삼각대 필요
- 촬영 모드에 따라 사용 불가능한 플래시 모드가 있음 → [[촬영모드-비교]] 참조
- 연속 촬영(Sequential High) 모드에서는 플래시 비활성화

## 플래시 강도 조절 (Flash Intensity Control)

- [[노출보정]]과 별도로 플래시 출력만 미세 조정 가능
- TG-7: Custom Menu D > **7+F** 옵션을 [On]으로 설정하면 노출 보정값이 플래시 보정에도 합산됨
- TG-7: Custom Menu D > **A+WB** 옵션으로 플래시 사용 시 자동 화이트밸런스 전환 가능
