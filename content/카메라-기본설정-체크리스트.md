# 카메라 기본설정 체크리스트 (TG-6 / TG-7 공통)

> **핵심 takeaway**: 입수 전 보트 위에서 세팅을 완료하고, **물속에서는 노출 보정과 스트로브 파워만 조정**한다. 이 체크리스트를 외워두면 세팅 실수로 다이빙을 날리는 일이 없다.

## 입수 전 (보트 위) — Set & Forget

### 1단계: 리셋

- [ ] Shooting Menu 1(촬영 메뉴 1) → **Reset/Assign Custom Modes(재설정/사용자 정의 등록)** → **Setting Reset(세팅 리셋)**

### 2단계: 기본 설정

| 설정 | TG-7 (영어) | TG-6 (한글) | 값 |
|---|---|---|---|
| Picture Mode | Picture Mode | 화상효과설정 | **Natural(네추럴)** |
| Live View Boost | Custom Menu B > Live View Boost | 설정 B > 라이브 뷰 부스트 | **ON** (LCD 밝게) |
| ISO Auto 범위 | Custom Menu C > ISO-Auto Set | 설정 C > ISO 자동 | Upper: **400** / Default: **100** |
| 최소 셔터 속도 | Custom Menu C > Lowest S/S Setting | 설정 C > 최소 셔터 속도 | **1/125** |

### 3단계: 촬영 모드별 설정

**와이드 사진**: A 모드 / f/2.8 / 노출 -1.0~-2.0 / Fill-In → [[와이드-촬영-세팅]]
**마크로 사진**: A 모드 / f/9~f/18 / 노출 -2.0 / Fill-In → [[마크로-사진-세팅]]
**마크로 영상**: P 모드 / 노출 -0.7 / Flash Off → [[마크로-영상-세팅]]
**와이드 영상**: P 모드 / 노출 -0.7 / Flash Off → [[와이드-촬영-세팅]]

### 4단계: 빠른 메뉴 확인 (OK 버튼 → 라이브 컨트롤)

- [ ] ISO → **Auto** 확인
- [ ] WB(화이트밸런스) → **Auto** 확인 (스트로브 백색광이면 자동이 적합)
- [ ] 화면비 → **4:3** (사진) 또는 **16:9** (영상)
- [ ] 이미지 파일 → **L(큰 이미지) F(보통 압축) + RAW**
- [ ] AF 모드 확인

### 5단계: 플래시/조명 설정

- [ ] **I(오른쪽) 버튼** → 플래시 모드 선택
  - 스트로브 TTL → **Fill-In(강제 발광)**
  - 스트로브 Manual → **Manual Value (1/64)**
  - 비디오 라이트만 → **Off(발광 금지)**
- [ ] **G(아래쪽) 버튼** → **Single(단일 촬영)** 확인 (연사 시 스트로브 못 따라감)

## 입수 후 — 물속에서 조정

### 잊어야 할 것 (물속에서 건드리지 말 것)

보트에서 세팅한 ISO, 화이트밸런스(사진 시), 화질, 조리개 등은 **뇌에서 지우기**.

### 기억할 것 (2가지만)

#### 1. 노출 보정 — 배경 컨트롤

**F(위쪽) 버튼** → HI 또는 컨트롤 다이얼

- 줄이면(-) → **더 파란(어두운) 배경** → 피사체 분리 쉬움
- 올리면(+) → **밝은 배경**
- 범위: **-1.0 ~ -2.0** (와이드), **-2.0** (마크로) 상황 보고 조절

#### 2. 스트로브/라이트 — 전경 컨트롤

- 파워 조절: 피사체 밝기 조정
- 위치 조절: 그림자/하이라이트 방향
- → [[스트로브-배치와-운용]] 참조

## C1/C2에 저장하기

자주 쓰는 세팅을 완성했다면:
1. 메뉴 → Shooting Menu 1(촬영 메뉴 1)
2. **Reset/Assign Custom Modes(재설정/사용자 정의 등록)**
3. **C1**: 마크로 세팅 저장
4. **C2**: 와이드 세팅 저장

→ 모드 다이얼을 C1/C2로 돌리면 즉시 불러오기
